# Mouse Gestures like Opera
A Mouse Gestures script is the same as in the old Opera

# Default settings
- On page:
  - ↑ : Scroll up
  - ↓ : Scroll down
  - ← : Back
  - → : Forward
  - ↑↓ : Reload
  - ↑→ : New tab
  - ↓↑ : Duplicate Tab
  - ↓→ : Close tab
  - →↑ : Scroll to top
  - →↓ : Scroll to bottom
  - ↑↓↑ : Reload without cache
- On link:
  - ↓← : Open link in new tab
  - ↓←↓ : Open link in new background tab
